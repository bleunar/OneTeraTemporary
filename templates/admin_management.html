Admin Roles & Permissions

Super Admin

Full Access

Complete system access with full administrative privileges including user management and system configuration.

Permissions:
  • Create and manage all admin accounts
  • View and manage all emergency reports
  • Send system-wide alerts
  • Access all dashboard features
  • System configuration and settings

Radio Operator

Dispatch

Emergency response coordination and dispatch management with real-time response tracking.

Permissions:
  • View all emergency reports
  • Update report status and dispatch responses
  • Calculate estimated arrival times
  • Access emergency hotlines
  • Coordinate with response teams

MDRRMO

Emergency Management

Municipal Disaster Risk Reduction and Management Office with full emergency management capabilities.

Permissions:
  • View and manage all emergency reports
  • Dispatch emergency responses
  • Send system-wide alerts
  • Update report status
  • Access all dashboard features
  • System configuration access

Mayor

View Only

View-only access for monitoring emergency situations and system statistics without modification capabilities.

Permissions:
  • View all emergency reports
  • Access system statistics
  • Monitor response activities
  • Read-only dashboard access

Engineer

Technical

Technical support and system maintenance access for engineering department requirements.

Permissions:
  • System maintenance access
  • Technical support features
  • Infrastructure monitoring

Barangay Admin

Local

Barangay-level administration access for local emergency management and community coordination.

Permissions:
  • Barangay-specific reports
  • Local emergency management
  • Community coordination

MSWDO

Social

Social welfare and development office access for humanitarian response and social services coordination.

Permissions:
  • Social services coordination
  • Humanitarian response
  • Welfare assistance management
